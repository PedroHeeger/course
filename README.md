## EM CONSTRUÇÃO

A organização dos arquivos de estudo são divididos em três repositórios no GitHub. O primeiro é o repositório **bootcamp**, ainda em construção, nele terá alguns arquivos de bootcamps que realizei. O segundo é o repositório **course**, que é este atual, nela terá alguns arquivos de cursos que realizei. Já o terceiro cujo o nome do repositório é **video**, também ainda em construção, conterá algumas aulas em vídeo que realizei.

Cada um desses três repositórios terá pastas de nome referente a plataforma em que realizei a aula (vídeo), o curso, ou o bootcamp, para o primeiro nível de pastas. Um segundo nível será criado para pastas que conterem muito material, sendo assim divididas por nome do software (python, power_bi, mysql, javascript, etc.). Por fim, o terceiro nível conterá a pasta, indicada com um número de ordem para controle, onde terá todo material da aula (vídeo), curso ou bootcamp que realizei. Nessa pasta pode ter outras subpastas específicas, além de arquivos todos referente a essa aula (vídeo), curso ou bootcamp específico.

Por enquanto, como este repositório e os outros dois estão em construção ainda, só está disponível nesse repositório a pasta da plataforma **DIO** (Digital Innovation One), com o curso **curso_066** (Desvendando a Ifood Dev Week).
